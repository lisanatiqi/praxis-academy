cara penginstallan gradle

1. buat folder di C dengan nama gradle
2. masukkan file download gradle yang telah di download
3. extract fil .zip gradle di dalam folder grandle yang ada di disk C
4. klik kanan pada my computer -> propeties -> Advanced System Settings -> Environmental Variables
5. pilih System Variables -> path -> New -> isi dengan " C:\Gradle\gradle-5.6.3\bin "
6. klik ok untuk save
